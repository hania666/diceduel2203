# Dice Duel — Solana Mobile Game

A fast-paced dice game built natively for Solana Seeker and Saga devices.

> Roll. Win. Dominate Solana.

## Download

- Saga Dapp Store: coming soon
- GitHub Releases: https://github.com/hania666/diceduel2203/releases/tag/v1.0.0

## Features

- 3 Bet Types: Over 3 (x2), Under 4 (x2), Exact Number (x6)
- Boss Rounds every 10 rolls with x1.5 multiplier
- Last Stand mode with x5 multiplier
- Double or Nothing after every win
- VS Bot duels
- Global leaderboard powered by Supabase
- Daily & weekly records
- XP system with 10 levels
- 14 achievements
- Daily missions
- 7 languages: English, Russian, Ukrainian, Spanish, Chinese, Korean, Japanese
- Mobile Wallet Adapter integration

## Build
```bash
git clone https://github.com/hania666/diceduel2203.git
cd diceduel2203
npm install
eas build --platform android --profile preview
```

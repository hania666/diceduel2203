# 🎲 Dice Duel — Solana Mobile Game


> **Roll. Win. Dominate Solana.**

A fast-paced dice game built natively for **Solana Seeker** and **Saga** devices. Connect your Solana wallet via Mobile Wallet Adapter and start rolling!

---


## 🎮 Game Features

### Core Gameplay
- **3 Bet Types** — Over 3 (x2), Under 4 (x2), Exact Number (x6)
- **Boss Rounds** — Every 10 rolls triggers a x1.5 multiplier round
- **Last Stand** — All-in mode with x5 multiplier when low on points
- **Double or Nothing** — After a win, double it or lose it all
- **VS Bot Mode** — Duel against AI opponents

### Progression System
- **XP & Levels** — 10 player levels with unique titles
- **14 Achievements** — Unlock badges for streaks, wins, and special events
- **Daily Missions** — 3 fresh missions every day with point rewards
- **Daily Bonus** — Login every day for increasing rewards

### Social & Competitive
- **Global Leaderboard** — Real-time rankings powered by Supabase
- **Daily & Weekly Records** — Top wins tracked by time period
- **Live Feed** — See other players' wins in real time
- **Player Profile** — Custom avatar and nickname

### Technical
- **7 Languages** — English, Russian, Ukrainian, Spanish, Chinese, Korean, Japanese
- **Splash Screen** — Animated launch screen
- **Offline Support** — Local progress saved via AsyncStorage
- **Optimized** — Request batching, 60s cache, 8s timeouts

---

## 🔗 Solana Integration

- **Mobile Wallet Adapter (MWA)** — Native wallet connection on Seeker/Saga
- **Devnet** — Currently running on Solana Devnet
- **Points Mode** — Virtual points (no real money in current version)
